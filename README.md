## Frontend Setup (Next.js + TailwindCSS)

### Requirements
- Node.js ≥ 18  
- npm (comes with Node)

---

### Installation
```bash
cd frontend
npm install
```

### Developemnt
```bash
npm run dev
```
- Open local host: http://localhost:xxxx
